# 🥗 Smart Dinner Suggester (夜ご飯最適化システム)

ユーザーの基本属性と昼食の内容に基づき、1日の理想的な栄養バランスを保つための「最適な夜ご飯」を提案するStreamlitアプリです。

## 📝 プロジェクト概要
「昼に何を食べたか」を入力すると、厚生労働省の基準に基づいた1日の推定エネルギー必要量から逆算し、その日の夜に摂取すべき栄養量とおすすめのメニューを提案します。

### 主な機能
- **パーソナライズ入力**: 年齢・性別・身体活動レベルに基づき基礎代謝量と必要エネルギーを算出。
- **目的別モード選択**: 「健康維持モード」「ダイエットモード」に合わせたカロリー調整。
- **夜ご飯の動的提案**: `推定必要量 - 昼食摂取量 = 残り許容量` をリアルタイムで計算。

## 🧮 計算ロジック
厚生労働省の基準に準拠した以下の計算式を使用しています。

- **推定エネルギー必要量 (kcal/日)** = 基礎代謝量（体重 × 基礎代謝基準値） × 身体活動レベル
- **目標夜ご飯カロリー** = 推定エネルギー必要量 - 昼食のエネルギー量

## 🛠️ 技術スタック
- **Python 3.12+**
- **Streamlit**: フロントエンドおよびUI
- **ハリス・ベネディクト方程式**: 基礎代謝量の算出
