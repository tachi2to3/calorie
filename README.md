# 🥗 Smart Dinner Suggester (夜ご飯最適化システム)

ユーザーの基本属性と夕方までの食事内容に基づき、1日の理想的な栄養バランスを保つための「最適な夜ご飯」を提案するStreamlitアプリです。

## 📝 プロジェクト概要
「朝食の有無」「食べた昼食」を入力すると、厚生労働省の基準に基づいた1日の推定エネルギー必要量から逆算し、その日の夜に摂取すべきカロリーを計算します。

### 実装済み機能
- **パーソナライズ入力**: 年齢・性別・身体活動レベルに基づき基礎代謝量と必要エネルギーを算出。

## 🧮 計算ロジック
厚生労働省の基準に準拠した以下の計算式を使用しています。
- **BMR（基礎代謝量）**
  1. 男性 = 13.397 * 体重 + 4.799 * 身長 - 5.677 * 年齢 + 88.362
  2. 女性 = 9.247 * 体重 + 3.098 * 身長 - 4.33 * 年齢 + 447.593
- **推定エネルギー必要量 (kcal/日)** = 基礎代謝量 × 身体活動レベル
- 身体活動レベル（5段階）
  1. "ほぼ運動しない (デスクワーク中心)": 1.2,
  2. "軽い運動 (週1〜3回程度)": 1.375,
  3. "中程度の運動 (週3〜5回程度)": 1.55,
  4. "激しい運動 (週6〜7回程度)": 1.725,
  5. "非常に激しい運動 (プロアスリート並み)": 1.9

- **目標夜ご飯カロリー** = 推定エネルギー必要量 - 昼食のエネルギー量

## 🛠️ 技術スタック
- **Python 3.12+**
- **Streamlit**: フロントエンドおよびUI
- **ハリス・ベネディクト方程式**: 基礎代謝量の算出
